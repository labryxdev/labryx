# Labryx - AI Workflow Automation Platform

Transform repetitive tasks into intelligent workflows with AI-powered automation.

## Quick Start

```bash
npm install -g labryx
labryx workflow --interactive
```

## Features

- **Interactive Workflow Builder** - Create workflows through guided prompts
- **Template Library** - Pre-built templates for common automation tasks  
- **AI-Powered** - Leverage LLMs for content generation and decision making
- **Multiple Triggers** - Schedule, webhook, or manual execution
- **Platform Integrations** - Connect to email, social media, CRM, and more

## Usage

### Create a Workflow
```bash
# Interactive mode
labryx workflow --interactive

# From template
labryx generate --type email
labryx workflow --file workflow.yml
```

### Available Templates
- Email automation
- Social media posting  
- Content generation
- Data analysis
- Custom workflows
